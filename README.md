# SpigotUHC

Easily run a UHC on your Spigot Server

## Prereqs

- Java 21
- Maven
- Git

## Getting started

1. Clone the repo

2. Run the following command to build the spigot server dev env
   
   ```
   REFRESH_BUILD=true ./setup_server.sh
   ```

   Note: This may take a long time (10-15min)

## Running

1. Run Spigot server

   ```
   ./run_server.sh
   ```
